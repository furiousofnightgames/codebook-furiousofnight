Exemplos e Explicação

  • Exemplo 1: Baixar Um PDF
  1. Crie uma pasta e coloque seus arquivos para download, nomeie de forma fácil de identificar.
  2. Dica: Nas aspas use o atalho que já foi ensinado nas aulas passadas,
    Dentro das aspas em href=" " segure a tecla CTRL e aperte a tecla ESPAÇO e escolha a pasta e seu arquivo com o conteúdo para download. O navegador talvez te dê opções para usar o PDF.

Exemplo de estrutura usada:

  • 
<a href="links-downloads/links_downloads.pdf" download="links_downloads.pdf" type="application/pdf"></a>
  • Usamos na estrutura o elemento 
    <a href="conteúdo"></a>
    para anexar o conteúdo para download.
  • Usamos também o elemento download="descrição" dentro do primeiro elemento para descrever o conteúdo.
  • E usamos também dentro do primeiro elemento o elemento type="application/pdf".
  • Observações importantes:
    - O atributo download sugere ao navegador que o arquivo seja baixado, em vez de aberto.
    - Nem todos os navegadores respeitam esse atributo para todos os tipos de arquivo.
    - Para maior compatibilidade, hospede seus arquivos em um servidor confiável
    - Caso nao faça o download do pdf use o botão direito do mouse e SALVAR LINK COMO o download do PDF terá sucesso.

Exemplo de estrutura usada:

  • <a href="links-downloads/links_downloads.zip" download="links_downloads.zip" type="application/zip"></a>
  • Oquê mudou?:
    Entre o primeiro e segundo exemplo é usamos os seguintes elementos:
    1. download="conteúdo.zip" com .zip na descrição do conteudo dentro do elemento download.
    2. type="application/zip" no elemento type="" usamos o application/zip para assim ter sucesso no download.
