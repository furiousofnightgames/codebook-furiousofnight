O que são links e âncoras?

<a> é a tag usada para criar links em HTML.
O atributo href define o destino do link.
Você pode usar links para navegar entre páginas, baixar arquivos ou ir para uma parte específica da mesma página (âncora).

Exemplo básico de link

<a href="https://www.exemplo.com" target="_blank">Clique aqui no Exemplo</a>

O texto entre <a> e </a> é o que aparece como link clicável.

Links externos

Links externos levam o usuário para outro site. O recomendado é usar target="_blank" para abrir em nova aba e rel="external" para segurança e semântica: 

<a href="https://www.google.com" target="_blank" rel="external">Google</a>

Dica: Sempre use https:// para links externos seguros.
Importante: rel="external" indica que o link é para fora do seu site.
Você pode adicionar junto de external o atributo "external noopener noreferrer" protege contra riscos de segurança.

Dicas e boas práticas

  • Sempre use title="abrir nova aba" para descrever o destino do link (acessibilidade).
  • Evite usar “Clique aqui” como texto do link. Prefira textos descritivos.
  • Links internos facilitam a navegação em páginas longas.
  • Use target="_blank" e rel="external noopener noreferrer" em links externos para segurança e semântica.

Exemplo de estrutura

código dos links externos: usando title="" & rel="" 

<a href="https:sua-URL" target="_blank" title="abrir nova aba">descrição</a>
    
<a href="https://www.sua-URL" target="_blank" rel="external">descrição</a>
